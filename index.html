Gesprekken

Gesprek 1 Cyd

Mijn leerdoelen tijdens de minor Web Development zijn het leren hoe samenwerking en tools binnen webontwikkeling werken, het verbeteren van mijn vaardigheden in CSS animaties en interactieve vormgeving en het ontwikkelen van websites die optimaal responsief zijn voor verschillende apparaten.

Tijdens deze minor wil ik mijn technische skills en programmeerkennis verder ontwikkelen tot een niveau waarop ik zelfstandiger en zelfverzekerder code kan schrijven. Ik wil mijn leerdoelen behalen door actief samen te werken, bijvoorbeeld tijdens het hackathonvak. Daar wil ik leren van andere en meer ervaren studenten over hoe zij technisch samenwerken aan een website en welke uitdagingen daarbij komen kijken.

Daarnaast vind ik CSS erg interessant en wil ik mij verder verdiepen in het maken van animaties en interactieve elementen. Ook wil ik mijn kennis uitbreiden in het ontwikkelen van responsieve websites, zodat deze goed werken op verschillende apparaten.

Gesprek 2 Vasilies

Mijn nieuwe leerdoelen voor de meesterproef zijn het beter begrijpen van JavaScript. Dit wil ik bereiken door veel JavaScript termen op te zoeken en de betekenis en functies daarvan beter te begrijpen.

Daarnaast wil ik beter leren samenwerken in GitHub met een team. Omdat ik eerder een keer in GitHub heb samengewerkt, vind ik het soms nog lastig, vooral wanneer er mergeconflicten ontstaan. Dit geeft mij vaak stress en zorgt ervoor dat ik niet goed weet hoe ik verder moet. Dit leerdoel wil ik behalen door bij problemen hulp te vragen aan medestudenten en wanneer nodig aan docenten.

Mijn derde leerdoel is het maken van toegankelijke en gebruiksvriendelijke websites voor de doelgroep. Tijdens de minor hebben we meerdere vakken gehad die gericht waren op UX zoals HCD en Browser Technology. Toch wil ik mij verder verdiepen in het ontwikkelen van toegankelijke websites, bijvoorbeeld voor screenreaders en leren hoe code op een manier geschreven wordt die voor zoveel mogelijk gebruikers toegankelijk is.
